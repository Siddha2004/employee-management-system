# Employee Management System

## 🚀 Tech Stack
- Frontend: React (Vite)
- Backend: Spring Boot
- Database: H2 / MySQL

## ✨ Features
- Add Employee
- View Employees
- Update Employee
- Delete Employee

## ▶️ Run Project

### Backend
cd backend
mvn spring-boot:run

### Frontend
cd frontend
npm install
npm run dev

## 🌐 URLs
- Frontend: http://localhost:5175
- Backend: http://localhost:8081
